# Traffic Copilot AI

AI SaaS de navegação inteligente com suporte a:
- Driver Mode (motoristas)
- Fleet Mode (empresas de delivery)
- Emergency Mode (ambulância/polícia/bombeiros)

## Objetivo do MVP
- Mapa + rota + trânsito (via API)
- Chat com agente (texto)
- Sugestão simples de rota e alertas
- Telemetria básica (eventos)
